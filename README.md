# SD Miniprojekt 1 - Lista

Starter project in C++ for:
- ArrayList
- SinglyLinkedList
- DoublyLinkedList
- menu for manual testing
- benchmark saving CSV files to `results/`

## Build (g++)

```bash
g++ -std=c++17 source_files/*.cpp -o sd_project
```

## Run

```bash
./sd_project
```

## Notes

- structures are implemented manually, without STL containers
- benchmark results are saved as CSV files
- add your own report and graphs based on generated CSV files
